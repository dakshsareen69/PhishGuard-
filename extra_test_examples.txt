============================================================================
  HOW TO TEST MANUALLY
============================================================================

  Via the UI:
    1. Open http://localhost:5000 in your browser
    2. Paste any URL into the URL tab, or text into the Text tab
    3. Click "Scan" and observe the risk score and rank

  Via the API (PowerShell):
    Invoke-RestMethod -Method POST -Uri http://localhost:5000/analyze `
      -ContentType "application/json" `
      -Body '{"type":"text","content":"YOUR TEXT HERE"}'

  Via the API (curl):
    curl -X POST http://localhost:5000/analyze \
      -H "Content-Type: application/json" \
      -d '{"type":"text","content":"YOUR TEXT HERE"}'
